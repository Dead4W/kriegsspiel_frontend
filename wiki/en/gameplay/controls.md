# Controls

Quick reference for core controls in a battle room.

## Camera and map

- **W / A / S / D** or **Arrow keys** — move the camera.
- **Mouse wheel** — zoom in/out (zoom follows cursor position).
- **RMB + mouse move** — drag the camera when no units are selected.

## Unit selection

- **LMB on a unit** — select a unit.
- **LMB + drag on empty area** — box select.
- Hold **Shift** while clicking/boxing to add units to current selection.
- Hold **Ctrl** while clicking/boxing to remove units from current selection.
- **Double click on a unit** — select similar units inside current view.

## Movement and commands

- **LMB + drag selected units** — move selected units (normal mode).
- For the umpire during war stage, dragging can place move commands (depends on hard move behavior).
- **Delete** — remove selected units.

## Unit rotation

- **Shift + Q / E** — rotate selected unit left/right.
- **Drag + mouse wheel** — rotate while dragging (with exactly one selected unit).
- For umpire during war stage, wheel rotation while dragging uses **Shift + Drag + mouse wheel**.

## UI hotkeys

- **V** — toggle ruler tool.
- **Esc** — close active tool.
