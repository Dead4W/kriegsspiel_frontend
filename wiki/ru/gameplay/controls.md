# Управление

Краткая памятка по основному управлению в боевой комнате.

## Камера и карта

- **W / A / S / D** или **стрелки** — перемещение камеры.
- **Колесо мыши** — приближение и отдаление карты (зум в позицию курсора).
- **ПКМ + движение мыши** — перетаскивание камеры, если нет выделенных юнитов.

## Выделение юнитов

- **ЛКМ по юниту** — выбрать юнит.
- **ЛКМ + drag по пустому месту** — выделение рамкой.
- **Shift** при клике/рамке — добавить юниты к текущему выделению.
- **Ctrl** при клике/рамке — убрать юниты из текущего выделения.
- **Двойной клик по юниту** — выделить похожие юниты в видимой области.

## Перемещение и команды

- **ЛКМ + drag по выделенным** — переместить выделенные юниты (обычный режим).
- Для судьи в фазе войны drag может работать как постановка приказа на движение (зависит от режима hard move).
- **Delete** — удалить выбранные юниты.

## Поворот юнита

- **Shift + Q / E** — повернуть выделенный юнит влево/вправо.
- **Drag + колесо мыши** — поворот (когда выделен ровно один юнит).
- Для судьи в фазе войны поворот колесом во время drag: **Shift + Drag + колесо**.

## Быстрые клавиши интерфейса

- **V** — открыть/закрыть инструмент линейки.
- **Esc** — закрыть активный инструмент.
