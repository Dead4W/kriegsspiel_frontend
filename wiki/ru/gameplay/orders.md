# Судья - Приказы

Приказы — это основной способ управления выбранными юнитами в бою.

## Кто может отдавать приказы

- Панель приказов доступна стороне **Судья**.
- В боевой комнате выделите юнитов и используйте кнопки приказов внизу экрана.

## Стандартный порядок действий

1. Выберите юнита или группу с одной стороны.
2. Откройте нужный приказ (кнопкой или контекстным действием).
3. Задайте цели и параметры.
4. Подтвердите через **Применить** (или **E / Enter**).

**Q / Esc** закрывает активный приказ без применения.

## Кнопки приказов и горячие клавиши

- **1** — переключить **автоатаку** у выбранных юнитов.
- **2** — **Атака** (выбор вражеских целей, поддерживает модификаторы способностей).
- **3** — **Смена формации**.
- **4** — **Ожидание** (задержка и необязательный комментарий).
- **5** — **Отступление** (задаёт длительность и заменяет текущую очередь).
- **+ / -** — увеличить/уменьшить мораль выбранных юнитов.
- **C** — очистить запланированные команды выбранных юнитов.

## Контекстные приказы (правая кнопка мыши)

- **ПКМ по пустому месту** — открыть/продолжить маршрут **движения**.
- **ПКМ по вражескому юниту** — открыть **атаку** для выбранной группы (если хотя бы один атакующий в радиусе).
- **ПКМ посыльными по союзному юниту** — открыть **доставку**.

### Детали маршрута движения

- Добавляйте точки маршрута через ПКМ.
- Для каждой точки можно выбрать модификатор среды (радиальное меню с иконками).
- Флаг **Патруль** делает маршрут циклическим.
- Подтвердите маршрут кнопкой **Применить**.

## Заметки

- Приказы ставятся в очередь, поэтому можно планировать поведение на несколько шагов.
- Атака работает параллельно: юнит может двигаться и атаковать одновременно.
- Атака, ожидание, движение и другие команды могут идти последовательно.
- Если план изменился, быстро очищайте очередь через **C**.
