# Судья - Уведомления

Панель уведомлений помогает **Судье** быстро находить юниты, которым нужно внимание.

## Где отображается

- Видна только для роли **Судья**.
- Скрывается после перехода боя в стадию **End**.
- Расположена в правом нижнем углу боевого интерфейса.

## Типы уведомлений

### Юниты с новым приказом

- Показывает юнитов, у которых появились новые назначенные команды.
- Удобно для быстрой проверки последних изменений приказов.

### Юниты без приказа

- Показывает юнитов без активной очереди команд.
- Автоматическое отступление не считается здесь обычным запланированным приказом.

### Юниты без боеприпасов

- Показывает юнитов с нулевыми патронами, которые не находятся в отступлении.
- Появляется только при включенной настройке комнаты **Патроны не бесконечные**.

### Посыльные без приказа

- Показывает посыльных без назначенной команды.
- Помогает не оставлять юнитов связи без задачи во время активной фазы.

## Что происходит при клике

Клик по любому уведомлению:

1. выделяет все юниты этой категории;
2. перемещает камеру в центр выбранной группы.

Так проще быстро проверять и закрывать пробелы в управлении во время боя.
