## `abilities`

Путь: `abilities.types[]`

Это способности юнитов. Они нужны, чтобы задавать особые режимы или действия, которые юнит может использовать в бою.

Например, способность может ускорять юнит, усиливать атаку или задавать параметры для неточного огня по области.

---

## ПАРАМЕТРЫ

Каждый элемент в `abilities.types[]` поддерживает:

- `id` (обязательно, string) - уникальный ID способности
- `title` (опционально, string) - отображаемое имя способности
- `multipliers` (опционально, object) - модификаторы характеристик, пока способность активна
- `params` (опционально, object) - дополнительные параметры, специфичные для способности

В `multipliers` можно использовать стандартные характеристики:

- `damage`
- `takeDamageMod`
- `speed`
- `attackRange`
- `visionRange`
- `fatigue` (только для способностей)

Все значения должны быть числовыми.

В `params` поддерживается:

- `radius` (number) - множитель радиуса для неточного огня по области

---

## ФОРМАТ `multipliers`

`multipliers` - это модификаторы, которые действуют, пока способность активна.

```json
{
  "multipliers": {
    "speed": 1.5,
    "fatigue": 2
  }
}
```

Например:

- `speed: 1.5` увеличивает скорость до 150%
- `fatigue: 2` увеличивает накопление усталости в 2 раза

---

## ФОРМАТ `params`

`params` нужны для дополнительных настроек способности.

```json
{
  "params": {
    "radius": 1.2
  }
}
```

Например, `radius: 1.2` увеличивает радиус неточного огня на 20%.

---

## ПРИМЕР

```json
{
  "abilities": {
    "types": [
      {
        "id": "charge",
        "title": "Charge",
        "multipliers": {
          "speed": 1.5,
          "fatigue": 2
        }
      },
      {
        "id": "inaccuracy_fire",
        "title": "Area Fire",
        "params": {
          "radius": 1.2
        }
      }
    ]
  }
}
```

---

## ОБЪЯСНЕНИЕ ПРИМЕРА

- `charge` - способность рывка или натиска.
- `speed: 1.5` означает, что при ее использовании юнит двигается быстрее.
- `fatigue: 2` означает, что за это ускорение юнит быстрее устает.
- `inaccuracy_fire` - способность неточного огня по области.
- `radius: 1.2` означает, что радиус такого удара будет больше обычного.

---

## ДОПОЛНИТЕЛЬНО

- ID способностей в `units.types[].abilities` используются как обычные ID.
- Если способность указана у юнита, но не описана в `abilities.types[]`, для нее не будут применяться специальные модификаторы и параметры из этого раздела.
