## `distanceModifiers`

Путь: `distanceModifiers`

Этот раздел задает, как меняется урон в зависимости от дистанции стрельбы.

С его помощью можно настроить:

- одну общую таблицу для всех юнитов через `default`
- отдельные таблицы для конкретных типов юнитов, например `artillery`, `cavalry` и других

---

## ПАРАМЕТРЫ

`distanceModifiers` - это объект, где каждый ключ содержит таблицу дистанций для определенного типа юнита.

Пример:

```json
{
  "distanceModifiers": {
    "default": [{ "distance": 200, "modifier": 1.0 }],
    "artillery": [{ "distance": 800, "modifier": 1.0 }]
  }
}
```

Каждая таблица представляет собой массив точек. У каждой точки есть два поля:

- `distance` (обязательно, number) - дистанция в метрах, должна быть `>= 0`
- `modifier` (обязательно, number) - множитель урона на этой дистанции, должен быть `> 0`

Если в таблице есть некорректные точки, они будут отброшены. Корректные точки автоматически сортируются по дистанции.

---

## КАК ВЫБИРАЕТСЯ ТАБЛИЦА

Когда юнит атакует, игра ищет подходящую таблицу в таком порядке:

1. Таблица, имя которой точно совпадает с типом юнита.
2. Таблица `default`.
3. Если не найдено ничего, используется множитель `1`.

Это позволяет задать общие правила для всех, а при необходимости переопределить их только для отдельных юнитов.

---

## КАК СЧИТАЕТСЯ МОДИФИКАТОР

- Если дистанция меньше или равна первой точке, берется `modifier` первой точки.
- Если дистанция находится между двумя точками, игра вычисляет промежуточное значение линейной интерполяцией.
- Если дистанция больше последней точки, используется экспоненциальный спад.

Для дистанций дальше последней точки действует отдельное правило:

- каждые дополнительные 100 метров уменьшают последний множитель в 2 раза
- формула выглядит так: `last.modifier / 2^steps`

Это значит, что после выхода за пределы таблицы урон продолжает падать все сильнее.

---

## ПРИМЕР

```json
{
  "distanceModifiers": {
    "default": [
      { "distance": 50, "modifier": 1.4 },
      { "distance": 250, "modifier": 1.0 },
      { "distance": 1000, "modifier": 0.2 }
    ],
    "artillery": [
      { "distance": 700, "modifier": 1.0 },
      { "distance": 4000, "modifier": 0.05 }
    ]
  }
}
```

Что означает этот пример:

- До `50` метров обычные юниты наносят `140%` урона.
- На `250` метрах урон становится обычным, то есть `100%`.
- К `1000` метрам урон падает до `20%`.
- Для `artillery` используется отдельная таблица, поэтому артиллерия считается по своим дистанциям, а не по `default`.
