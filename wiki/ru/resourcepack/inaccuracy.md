## `inaccuracy`

Путь: `inaccuracy`

Этот раздел задает неточность для способностей.

Например, его можно использовать для артиллерийской способности, которая бьет не точно в одну точку, а с разбросом по области.

Проще говоря: чем выше неточность, тем дальше фактическое попадание может оказаться от выбранной точки.

На неточность влияют:

- расстояние до цели
- разница по высоте между атакующим и целью

---

## ПАРАМЕТРЫ

Раздел `inaccuracy` поддерживает:

- `heightFactor` (опционально, number) - насколько сильно разница высот влияет на неточность
- `distanceFactor` (опционально, number) - насколько сильно расстояние до цели влияет на неточность

Если значение не указано или указано неправильно, используются стандартные значения:

- `heightFactor`: `5.0`
- `distanceFactor`: `0.1`

---

## ФОРМАТ

```json
{
  "inaccuracy": {
    "heightFactor": 5.0,
    "distanceFactor": 0.1
  }
}
```

Оба параметра должны быть числовыми.

---

## ПРИМЕР

```json
{
  "inaccuracy": {
    "heightFactor": 6.0,
    "distanceFactor": 0.15
  }
}
```

---

## ОБЪЯСНЕНИЕ ПРИМЕРА

- `distanceFactor: 0.15` увеличивает неточность на большой дистанции.
- `heightFactor: 6.0` усиливает влияние разницы высот на неточность.
- Такой набор подходит, например, для артиллерийской способности с ударом по области, где результат не должен быть идеально точным.
