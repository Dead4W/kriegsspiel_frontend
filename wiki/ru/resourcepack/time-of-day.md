## `timeOfDay`

Путь: `timeOfDay.segments[]`

Это времена суток. Они нужны для модификаторов времени суток, например чтобы уменьшить обзор ночью.

Диапазоны работают как полуоткрытый интервал: `[start, end)`.

Пример: `start = 9`, `end = 14` означает, что диапазон начинается в `09:00` и заканчивается в `13:59`.

---

## ПАРАМЕТРЫ

Каждый элемент в `timeOfDay.segments[]` поддерживает:

- `id` (обязательно, string) - уникальный ID сегмента (`day`, `night` и т.д.)
- `title` (опционально, string) - отображаемое имя сегмента
- `start` (обязательно, number) - час начала сегмента
- `end` (обязательно, number) - час конца сегмента
- `multipliers` (опционально, object) - модификаторы, которые применяются в этом сегменте

Дополнительно по диапазонам:

- `start < end` - обычный диапазон (внутри одного дня)
- `start > end` - диапазон через полночь
- `start === end` - сегмент активен всегда

---

## ФОРМАТ `multipliers`

`multipliers` - это объект групп с числовыми модификаторами характеристик.

```json
{
  "multipliers": {
    "default": {
      "visionRange": 0.7
    }
  }
}
```

Внутри группы можно указывать стандартные параметры модификаторов, например:
`damage`, `takeDamageMod`, `speed`, `attackRange`, `visionRange`.

---

## ПРИМЕР

```json
{
  "timeOfDay": {
    "segments": [
      {
        "id": "day",
        "title": "Day",
        "start": 8,
        "end": 20,
        "multipliers": {
          "default": {}
        }
      },
      {
        "id": "night",
        "title": "Night",
        "start": 20,
        "end": 8,
        "multipliers": {
          "default": {
            "visionRange": 0.6,
            "attackRange": 0.85
          }
        }
      }
    ]
  }
}
```

---

## ОБЪЯСНЕНИЕ ПРИМЕРА

- Сегмент `day` активен с `08:00` до `19:59`, модификаторов нет.
- Сегмент `night` активен с `20:00` до `07:59` (через полночь).
- Ночью `visionRange: 0.6` уменьшает обзор до 60% от базового.
- Ночью `attackRange: 0.85` уменьшает дальность атаки до 85% от базовой.
